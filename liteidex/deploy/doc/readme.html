Introduction

LiteIDE is a simple, open source, cross-platform IDE.

  • Project: LiteIDE
  • Version: X8 (1.25)
  • Author : visualfc
  • Website: code.google.com/p/golangide
  • Source : code.google.com/p/liteide
  • Group : liteide-dev

Features

  • Plugins extension system
  • Mime type based system
  • Multi environment system
  • Multi build system
  • Kate syntax and style scheme
  • WordApi complete helper
  • Golang helper astview gocode and godoc

System

  • Windows
  • Linux
  • MacOSX
