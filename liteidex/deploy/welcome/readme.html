Introduction

LiteIDE is a simple, open source, cross-platform IDE.

  • Project: LiteIDE
  • Version: X14.1 (2.1875)
  • Author : visualfc
  • Website: code.google.com/p/golangide
  • Source : code.google.com/p/liteide
  • Group : liteide-dev

System

  • Windows
  • Linux x86
  • Linux x86_64
  • MacOS X 10.6

Base Features:

  • Mime type basis system
  • System environment management
  • Project management
  • Build system management
  • Debug system simple and open
  • Kate syntax and style scheme
  • WordApi complete helper
  • Project and file wizard

Golang Support:

  • Golang package management
  • Golang package wizard
  • Golang package build
  • Golang std api serach
  • Golang ast browser
  • Golang doc browser
  • Gocode helper
